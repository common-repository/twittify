Twittify 

== Description ==

This plugin simply turns Twitter @usernames and #hashtags into links to the related place on Twitter.

Usernames are automatically linked to the user's profile page, and hashtags are linked to the Twitter Search page with those results.

== Installation ==


1. Upload `twittify.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
